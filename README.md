# 📱 Pesquisador de Contatos

Aplicação web simples desenvolvida para pesquisar contatos através do nome e retornar o telefone correspondente.

O projeto foi desenvolvido utilizando **HTML, CSS e JavaScript puro**, com foco em praticar manipulação do DOM, lógica de programação e organização de código no desenvolvimento Front-End.

---

## 🚀 Funcionalidades

- 🔎 Pesquisa de contatos pelo nome
- 📞 Exibição do telefone do contato encontrado
- ❌ Mensagem de contato não encontrado
- 🎨 Interface moderna e responsiva
- ⚡ Busca dinâmica utilizando JavaScript
